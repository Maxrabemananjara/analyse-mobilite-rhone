# Identite du projet

## Nom publie

**Analyse Mobilite Rhone**

J'ai choisi ce nom pour sortir d'un cadrage trop centre sur Lyon intra-muros ou sur un acteur precis. Le projet analyse la mobilite ouverte a l'echelle du Rhone et de la Metropole de Lyon, avec un usage metier oriente decision operationnelle.

## Positionnement

Projet portfolio data/BI :

- collecte et controle de donnees publiques ;
- transformation Python ;
- modelisation en constellation pour Power BI ;
- dashboard de synthese ;
- lecture metier orientee deploiement et couverture territoriale.

## Sous-titre

**Analyse open data de la demande de mobilite pour orienter des decisions de deploiement territorial.**
